# Finding The Best Camera Overall
The objective is to find a camera that is the best overall in terms of its specs. This uses a dataset that contains 1000 rows (each row a different camera) and 13 columns giving the specs of each of the camera.![Intro Background Slide](https://github.com/user-attachments/assets/8f5b0b6a-3fe6-4bc7-80dc-e2a40079b2fb)
